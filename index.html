Sobre mim

Meu primeiro contato com a programação foi em 2018, minha namorada que já conhecia bastante sobre o assunto me mostrou Python, e foi ai que minha paixão pela tecnologia começou, eu tenho o raciocinio logico muito bom, então logo comecei a aprender cada vez mais e a fazer meus mini programinhas, como calculadora com Bhaskara, jogo da forca, entre outros projetos basico.
Como eu sou curioso, comecei a pesquisar outras ferramentas com interface grafica, foi ai que conheci Flutter e Dart, e foi ai que minha mente explodiu, descobri que tem um mundo inteiro atrás de um aplicativo, front-end, back-end e milhares de formas de fazer os dois, então fui eu lá começar estudar outras ferramentas e a da vez agora foi o Sql Server, que é um banco de dados, uma das coisas obrigatorias em quase todas as aplicações para faze-las funcionarem de verdade.
Mas faltava algo, foi ai que conheci a Web, ai pronto, mais uma explosão na minha mente, conhecendo como funciona essas aplicações, oque está por trás. Html5, Css3 e Javascript as combinações mais perfeitas que conheci, que é onde eu estou estudando atualmente, aprendendo cada vez mais, e continuando conhecendo outras ferramentas como ReactJs.

Conhecimentos

Aqui estão alguns conhecimentos adquiridos com meus estudos.

Html 5

Linguagem de Marcação de HiperTexto, é o bloco de construção mais básico da web.

CSS 3

Usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

javascript

Linguagem de programação que permite a você implementar itens complexos em páginas web.

React js

React. js é, portanto, uma biblioteca de JavaScript, open source, utilizada para construir user interfaces, nomeadas para aplicações de página única. O diferencial do React é ser rápida, escalável e simples.

Flutter

O Flutter é uma biblioteca que utiliza a linguagem Dart da Google para criação de aplicações móveis Cross Platform.

Sql Server

Um dos recursos mais conhecidos do mundo, a linguagem SQL (Structured Query Language) é usada para executar comando em bancos de dados relacionais, isto é, baseado em tabelas.
